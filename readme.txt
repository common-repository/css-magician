== Installation ==
1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Css Magician Tab to configure Css Magician


== Frequently Asked Questions ==
**Do I need a special theme to use Css Magician**
No, Css Magician work for all the themes.
**Do I need to know how to code to use Css Magician**
Absolutely not! Css Magician is a live frontend configurator, editor and page builder. No need to be an advanced user to use it. You can update your theme without to write a line of Css Code.
**My site is working with WordPress 3.0, will Css Magician work for me?**
No. Css Magician need WordPress 5.0 or greater, and is compatible with PHP 5.6 or greater.
